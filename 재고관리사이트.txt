1.저장, 추가 기능에 개발 방향성
- 저장기능 : 각 셀은 변경 입력이 가능하도록 하고 이를 ctrl + s를 누르면 해당 변경사항이 저장(이는 key 이벤트로 구현)
- 추가기능 : 표 상단의 form을 통해 추가
- 월별 선택 기능 : 표 바로 위의 1월 ~ 12월의 탭과 마지막에 통계 탭을 추가
각 탭을 클릭하면 해당하는 내용을 출력 
2. 저장 구조에 대한 방향성
- 현재 배열로 구성
id : 각 원소별로 구별할 요소
pName : 상품명
days : 일 수별 
uPrice : 단가
price : 금액(세금포함)
vat : 부가세
// 총괄 금액은 갯수와 금액을 곱해서 출력

재고관리 현재 구조
id : 월별 id
selMonth : 현재 월
invList : 월별 재고 정보
-> 현재 문제 : 하나의 상품의 정보 밖에 못들어가는 상태
해결방법 -> []로 한번 더 싼다.
